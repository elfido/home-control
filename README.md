# projects-template
A combo of NodeJS + Express + Swig; jQuery + Bootstrap + CanJS + Underscore

##Dependencies
Grunt (install it globally with "sudo npm install -g grunt-cli")
Bower (install it globally with "sudo npm install -g bower")

##Init
Only the first time:
* npm install
* bower install
* grunt

Then to execute:
node index
